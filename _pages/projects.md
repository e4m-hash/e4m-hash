# Projects

## Featured Project

### FunOMIC2 Nextflow Pipeline
**기존 메타게놈 분석 파이프라인의 Nextflow 재구현**

기존에 존재하던 [FunOMIC2](https://github.com/ManichanhLab/FunOMIC2) 파이프라인을 Nextflow로 재패키징하여 재현성과 확장성을 개선한 프로젝트입니다.

#### Key Features
- **재현 가능한 워크플로우**: Nextflow DSL2 기반 모듈화된 파이프라인
- **컨테이너화**: Docker 기반 환경 격리 및 배포
- **확장성**: 클러스터 및 클라우드 환경 지원
- **모듈화**: 각 분석 단계별 독립적 모듈 구성

#### Technologies Used
- **Nextflow**: 워크플로우 관리
- **Docker**: 컨테이너화
- **Conda**: 패키지 관리
- **Bash/Python**: 스크립팅

#### Impact
- 기존 수동 실행 방식 대신 자동화
- 분석 재현성 보장
- 다양한 HPC 환경에서 호환성 확보

[📁 Repository 보기](https://github.com/e4m-hash/FunOMIC2-nextflow)
